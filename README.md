# Overview

English | [中文README](README.zh_CN.md)

HiAgent-Java-SDK is the Java SDK of the HiAgent product from Volcano Engine. Developers can use this SDK to quickly develop
functions and improve development efficiency. HiAgent-Java-SDK provides a complete AI native application development suite,
including a rich set of development components and application example code.
